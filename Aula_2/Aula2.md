# Introdução ao Git e ao GitHub

## Introdução ao controle de versão

O Git é um sistema de controle de versão distribuído, amplamente utilizado em projetos de software e, cada vez mais, em projetos científicos, como na área de bioinformática.
Ele permite que os desenvolvedores rastreiem as alterações em seus arquivos ao longo do tempo, revertam para versões anteriores, colaborem com outros desenvolvedores e gerenciem diferentes linhas 
de desenvolvimento (chamadas de "*branches*").

Já o GitHub é uma plataforma online que utiliza o Git como ferramenta de controle de versão, oferecendo recursos adicionais como armazenamento de repositórios, colaboração entre desenvolvedores, gerenciamento de projetos e integração com outras ferramentas.
