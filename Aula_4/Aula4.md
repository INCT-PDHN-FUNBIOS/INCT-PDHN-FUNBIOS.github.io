# Introdução ao Python

A primeira parte do tutorial introdutório à linguagem de programação Python foca em como estruturar a lógica de programação (variáveis, condicionais, repetições, funções) e como usar o pacote `numpy` para operações numéricas e o `pandas` para manipulação de dados. O tutorial é estruturado em pequenos blocos de código que podem ser executados diretamente no Google Colab. O objetivo é fornecer uma base sólida para que os participantes possam utilizar Python em suas análises, especialmente em bioinformática e estatística.
